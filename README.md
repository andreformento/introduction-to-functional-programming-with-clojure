# Introduction to functional programming with Clojure

Install clojure:
```shell
mkdir -p ~/bin
curl 'https://raw.githubusercontent.com/technomancy/leiningen/stable/bin/lein' > ~/bin/lein
chmod +x ~/bin/lein
echo 'export PATH=~/bin/lein:$PATH' >> /home/andre/.zshrc
```

Repl: `lein repl`
Create project: `lein new app forca`
